# Repository name

This repository is part of the ConvaSimu reproducibility package.

Related manuscript:
"Perceptual Fidelity in Driving Simulation Through Adaptive Rendering and Data-Driven Scene–Traffic Synthesis"

Main project repository:
https://github.com/TransLabConcordia/ConvaSimu

Contents:
- Source code
- Required dependencies
- Configuration files
- Sample inputs
- Reproduction instructions

Software/version information:
- Unreal Engine: 4.27
- Main implementation language: C++
- OpenDRIVE input: .xodr
- SUMO/NetEdit: used for OpenDRIVE preparation when maps are generated from OSM

How to use:
1. Clone or download the repository.
2. Open the Unreal Engine project/plugin.
3. Follow the sample input structure.
4. Run the provided example configuration.
5. Compare the generated output with the manuscript figures/tables.
