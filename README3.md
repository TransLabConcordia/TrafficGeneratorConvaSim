# ConvaSimu

ConvaSimu is a human-in-the-loop driving-simulation framework developed for the manuscript:

"Perceptual Fidelity in Driving Simulation Through Adaptive Rendering and Data-Driven Scene–Traffic Synthesis"

This repository provides the main access point for the ConvaSimu reproducibility package.

Modules:
1. OpenDRIVE-to-Unreal road-network generation:
   https://github.com/TransLabConcordia/ConvaSimuOpenDriveToMap

2. Trajectory-based traffic generation:
   https://github.com/TransLabConcordia/TrafficGeneratorConvaSim

The repositories include source code, configuration files, sample maps, representative trajectory inputs, and instructions needed to reproduce the main results reported in the manuscript.

Please cite the associated paper and this repository if you use this code.
